Die SQlite-Datenbank kann man entweder im Startfenster mittels Dateibrowser auswählen. Unsere Beispiel DB ist die hohoho.db.
Ansonsten können Sie im Startfenster einfach auf Standard klicken. 
Dies erstellt im Arbeitsverzeichnis, falls noch nicht bereits vorhanden, die hohoho.db und verwendet diese. 
Sollte die Standard hohoho.db bereits existieren so verwendet er diese und überschreibt Sie nicht!
